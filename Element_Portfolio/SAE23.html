SAE23 - Mettre en place une solution informatique pour l'entreprise

Description

La SAE23 a pour titre "Mettre en place une solution informatique pour l'entreprise". L'objectif était de comprendre et d'utiliser le langage JavaScript et l'environnement NodeJS afin de créer la partie Backend (interaction avec serveur) d'un site web de gestion de tickets. Ce travail était à faire en autonomie dans le cadre de mon deuxième semestre.

Travail Réalisé

Ce projet était dicté par les étapes disponibles sur le site "Moodle - cours en ligne". J'ai appris ce qu'était un site internet d'architecture full-stack (frontend et backend), l'utilisation de NodeJS, le langage JSON et la configuration et utilisation d'une base de données MongoDB. L'interface du site de gestion de ticket est visible avec l'image de droite et un extrait du cours sur le site "Moodle" est disponible avec l'image de gauche.

Résultat

Pour l'instant, je n'ai pas reçu de résultats, cependant lors des séances, j'ai pu comprendre ces nouvelles notions et je peux affirmer que ma compétence en "Programmation Web" a pu augmenter avec cette activité.
